# Source Map: Sgcarmart Assistant

## Tier 1 (primary)
- Official Sgcarmart listing pages (canonical listing URLs)
- Official listing seller/dealer profile details shown on Sgcarmart

## Tier 2
- Official dealer websites for warranty/contact verification
- Official LTA/OneMotoring references for regulatory clarification (when user asks transfer or ownership-process questions)

## Tier 3 (supporting only)
- Community forum posts or reposted listing screenshots

## Rules
- Use Tier 1 for core pricing/spec/seller claims whenever possible.
- If Tier 1 and Tier 2 conflict, prefer Tier 1 for listing facts and clearly flag the conflict.
- Never use Tier 3 alone for numeric claims.
- Include source links and absolute timestamps for time-sensitive statements.
- If a listing appears stale or unavailable, lower confidence and state limitation.
