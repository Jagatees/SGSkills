# SG Skill Creator Human Review Contacts

Use this file when output is `Human Review Required`.

## Roles
- Safety lead role: SGSkills Safety Lead
- Domain approver role: Domain Owner (for regulated domains)
- Release approver role: Repository Maintainer

## Escalation routing
- `HIGH` risk request: Safety Lead + Repository Maintainer approval required.
- Regulated domain request: Domain Owner + Safety Lead approval required.
- Election/sensitive-period high-impact claims: Safety Lead approval required before publish.

## Evidence required in escalation packet
- Request summary and intended skill behavior
- Risk tier rationale and misuse scenarios
- Refusal triggers and escalation triggers
- Source trust plan (Tier 1/2/3)
- Privacy, redaction, and data export controls
- Proposed `review_due` date and recertification cadence

- Date (YYYY-MM-DD):
- Reviewer role(s):
- Decision (approved/rejected/changes required):
