# SG Skill Creator Output Preview Modes

Use preview-first selection before generating full output.

## Modes

### 1) Quick
- Best for: fast checks and rough drafts.
- Output: concise summary + minimal file plan.
- Tradeoff: lower detail and fewer examples.

### 2) Standard
- Best for: normal skill creation and updates.
- Output: complete scaffold + references + checks.
- Tradeoff: moderate length.

### 3) Strict
- Best for: regulated or public-impact domains.
- Output: complete scaffold + expanded verification, safety, and escalation notes.
- Tradeoff: longer review cycle.

## Preview-first workflow
1. Show 2-3 mini previews (usually Quick, Standard, Strict).
2. Ask user to pick one mode.
3. Generate full output only for selected mode.
