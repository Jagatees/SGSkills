# Output Template

Use this format unless user asks otherwise.

## 1) Latest BTO Snapshot
- What was announced (or latest available official update)
- Key date(s) in `Day Month Year` format
- Confidence: `High|Medium|Low`
- Source link(s)

## 2) BTO Process (Simple Steps)
1. Check project details and eligibility
2. Submit application in official window
3. Wait for ballot result and queue position
4. Book flat if invited
5. Complete documents/financing steps
- Confidence: `High|Medium|Low`
- Source link(s)

## 3) What You Should Do Now
- 3-5 practical checks (documents, timeline, financing)
- Mention unknowns clearly if user-specific details are missing
- Confidence: `High|Medium|Low`
- Source link(s)

## 4) Caveat
- State that policies/timelines can change; confirm with latest official pages.
