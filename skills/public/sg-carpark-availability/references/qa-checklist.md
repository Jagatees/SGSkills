# QA Checklist: SG Car Park Availability

- [ ] `As of` timestamp is present in SGT.
- [ ] Scope (time, area/location, lot type) is explicit.
- [ ] Official source is used for lot-count claims.
- [ ] Snapshot timestamp and freshness status are included.
- [ ] Each result row includes carpark identifier and update time.
- [ ] Confidence labels are attached to major sections.
- [ ] Facts and inference are separated.
- [ ] No guaranteed-availability language appears.
- [ ] Verification pending note is present when freshness cannot be confirmed.
