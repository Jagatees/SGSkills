# Source Map: SG Car Park Availability

## Tier 1 (primary)
- data.gov.sg transport API (carpark availability): https://api.data.gov.sg/v1/transport/carpark-availability
- data.gov.sg dataset index: https://data.gov.sg/

## Tier 2
- Official notices on data/API outages, maintenance, or delayed updates from source operators.

## Tier 3 (supporting only)
- Media/community reposts of parking availability.

## Rules
- Never issue lot-count claims from Tier 3 alone.
- Prefer official API/dataset values when sources differ.
- Include direct links and absolute timestamps for major sections.
- If freshness/publication status is unclear or pending, lower confidence and state limitations.
