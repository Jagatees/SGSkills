# Intake: SG COE Bidding Results

## Core
- Skill slug: `sg-coe-bidding-results`
- One-line purpose: Provide Singapore COE bidding result summaries by category with official links and confidence labels.
- Target users: drivers, fleet planners, auto-market watchers, public users.
- Domain: Singapore transport policy and COE exercise reporting.
- Risk tier: `LOW`.

## Inputs and outputs
- Required input: COE results query.
- Optional input: target category, exercise period, comparison mode.
- Output format: structured brief with category table and timestamped sections.
- Confidence labels: required (`High`/`Medium`/`Low`).

## Sources and quality
- Tier 1: LTA/OneMotoring official COE result and quota publications.
- Tier 2: established SG media for contextual explanation only.
- Tier 3: forums/social sources (supporting only).
- Verification rule: all category values must map to official published data or be marked unavailable.

## Safety and governance
- No fabricated prices/quotas.
- No guaranteed outcome language for future bids.
- Escalate personalized financial/legal instructions framed as certainty.
