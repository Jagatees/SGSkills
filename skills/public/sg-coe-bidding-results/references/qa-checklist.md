# QA Checklist: SG COE Bidding Results

- [ ] `As of` timestamp is present in SGT.
- [ ] Exercise scope is explicit.
- [ ] Official LTA/OneMotoring source is used for core result values.
- [ ] Categories are clearly labeled and scoped (`A` to `E`) when available.
- [ ] Every numeric result field is sourced or explicitly marked `N/A`.
- [ ] Comparison calculations are correct and clearly defined.
- [ ] Confidence labels are attached to major sections.
- [ ] Facts and inference are separated.
- [ ] No guaranteed future-bid outcomes are stated.
- [ ] Caveats are present for stale/incomplete/conflicting data.
