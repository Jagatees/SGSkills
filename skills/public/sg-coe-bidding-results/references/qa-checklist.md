# QA Checklist: SG COE Bidding Results

- [ ] `As of` timestamp is present in SGT.
- [ ] Scope (latest/specific exercise and categories) is explicit.
- [ ] Official OneMotoring/LTA source is used for result claims.
- [ ] Category table includes A-E rows (or requested subset only).
- [ ] Premium/quota/bids values are copied from official source or marked pending.
- [ ] Publication status and basis are explicit.
- [ ] Confidence labels are attached to major sections.
- [ ] Facts and inference are separated.
- [ ] No bidding-guarantee/manipulation language appears.
- [ ] Verification pending note is present when publication cannot be confirmed.
