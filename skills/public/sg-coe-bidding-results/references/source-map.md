# Source Map: SG COE Bidding Results

## Tier 1 (primary)
- OneMotoring COE open bidding and results page: https://www.onemotoring.lta.gov.sg/content/onemotoring/home/buying/upfront-vehicle-costs/certificate-of-entitlement-coe/open-bidding-and-results.html
- LTA website (COE-related pages): https://www.lta.gov.sg/

## Tier 2
- Official OneMotoring/LTA notices on bidding exercise schedules, publication delays, and exceptions.

## Tier 3 (supporting only)
- Media summaries or community reposts of COE category values.

## Rules
- Never issue category-premium claims from Tier 3 alone.
- Prefer official OneMotoring/LTA pages when sources differ.
- Include direct links and absolute timestamps for major sections.
- If publication status is unclear or pending, lower confidence and state limitations.
