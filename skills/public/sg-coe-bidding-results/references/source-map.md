# Source Map: SG COE Bidding Results

## Tier 1 (primary)
- LTA / OneMotoring official COE result publications and quota notices.
- Suggested priority links:
  - `https://onemotoring.lta.gov.sg/content/onemotoring/home/buying/upfront-vehicle-costs/coe-open-bidding.html`
  - `https://onemotoring.lta.gov.sg/content/onemotoring/home/buying/upfront-vehicle-costs/current_coe.html`

## Tier 2
- Mainstream Singapore reporting for non-authoritative context and narrative.
- Official auto-industry statements for interpretation support only.

## Tier 3 (supporting only)
- Community/forum discussions and social commentary.

## Rules
- Never publish price/quota values from Tier 2 or Tier 3 without Tier 1 confirmation.
- Prefer latest timestamped official publication when sources differ.
- Include direct links and absolute timestamps in each major section.
- If official pages are unavailable or delayed, mark `Low confidence` and `Verification pending`.
