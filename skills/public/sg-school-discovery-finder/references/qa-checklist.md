# QA Checklist

Run before final output:

- Every candidate in the main set has an official source URL.
- Tags are only: Core match, Consider, Watchlist.
- Include and exclusion reasons are explicit and concise.
- No admission guarantee or admission-probability claims.
- Absolute dates are used where relevant.
- Timestamp is shown in `SGT`.
- Source links are full clickable URLs (`https://...`), not labels only.
- `Facts` and `Inference:` are clearly separated.
- Handoff payload is complete for the next skill.
- When required inputs are missing, output `Input Required` and do not produce a final ranked candidate set.
