# Source Map (School Discovery)

Prioritize sources in this order:

1. MOE official school listing/directory pages
- School names, levels/types, addresses
- Official program and support summaries where available

2. Individual school official websites
- Program details, CCA focus, support information

Rules:
- Use official primary sources for inclusion decisions.
- If a school detail cannot be verified, lower confidence and move to `Watchlist` when needed.
- Keep source URLs at item level for traceability.
