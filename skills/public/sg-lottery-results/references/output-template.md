# Output Template: SG Lottery Results

Use this structure unless user requests a different layout.

```md
As of: <YYYY-MM-DD HH:mm SGT>
Scope: <4D | TOTO | both> (<latest draw | specific draw date>)

4D:
- Draw date: <YYYY-MM-DD or "Not published yet">
- Draw number: <value if available>
- Winning numbers: <official values or "Pending publication">
- Confidence: <High|Medium|Low>
- Source: <https://...>

TOTO:
- Draw date: <YYYY-MM-DD or "Not published yet">
- Draw number: <value if available>
- Winning numbers: <official values or "Pending publication">
- Confidence: <High|Medium|Low>
- Source: <https://...>

Facts vs inference:
- Facts: <published winning numbers, draw date, draw number, timestamps>
- Inference: <whether latest draw appears published/pending and why>

Caveat:
- <results may still be pending publication or delayed by official updates>

Verification note (only when needed):
- <Verification pending due to unavailable/unclear publication status>
```
