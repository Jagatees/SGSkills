# Insight Rubric

Score each candidate insight on 1-5 scale:

- Impact: How many people/services in Singapore are affected?
- Timeliness: Is this recent enough to matter now?
- Reliability: Is the metric stable, complete, and directly sourced?
- Actionability: Can a team make a practical decision from this?

Total score:
- 16-20: Priority 1 (Top highlights)
- 11-15: Priority 2 (Main section)
- 4-10: Priority 3 (Watch next or omit)

Confidence mapping:
- High: metric reproducible, data quality acceptable, no conflicting source.
- Medium: metric reproducible but partial gaps or ambiguity.
- Low: weak data quality, unresolved inconsistency, or unclear period mapping.

Rules:
- Do not place `Low` confidence insights in top highlights.
- If two insights are similar, keep only the higher-scoring one.
