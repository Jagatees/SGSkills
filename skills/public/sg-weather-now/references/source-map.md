# Source Map: SG Weather Now

## Tier 1 (primary)
- Meteorological Service Singapore (MSS) official forecast/nowcast pages
- National Environment Agency (NEA) weather advisories and warnings
- Suggested priority links:
  - `https://www.weather.gov.sg/weather-forecast-2hrnowcast/`
  - `https://www.weather.gov.sg/weather-forecast-24hrforecast/`
  - `https://www.weather.gov.sg/weather-warning/`

## Tier 2
- Official transport/operator notices that reference weather impact
- Official government channels relaying weather advisories

## Tier 3 (supporting only)
- Community/social weather observations

## Rules
- Never issue warning-level conclusions from Tier 3 alone.
- Prefer the latest timestamped MSS/NEA source when sources differ.
- Include direct links and absolute timestamps for every major section.
- If a nowcast timestamp is stale for operational planning context, lower confidence and state limitation.
